# Herzinger AI Portfolio

Portfolio website for Andreas Herzinger - Freelance AI Engineer

## Project Structure

```
portfolio/
├── src/
│   ├── components/     # HTML component files
│   ├── styles/        # CSS files
│   ├── scripts/       # JavaScript modules
│   └── assets/        # Images and other assets
├── index.html         # Main HTML file
├── package.json       # Dependencies
└── README.md          # This file
```

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The site will be available at `http://localhost:3000`

## Features

- Dark/Light theme toggle
- German/English language toggle
- Responsive design
- Interactive chatbot
- Project filtering
- Smooth animations
